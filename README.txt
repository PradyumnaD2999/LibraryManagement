Project Setup:

1) In eclipse, select the parent folder of libraryApp directory as workspace.
2) Import libraryApp as an existing maven project.
3) In pgAdmin4, run the Sample Data.sql file to set up database.

Description:
-In this project we have implemented an online library management system using maven.
-There are two tyes of users: Members and Librarians.
-Members can search for, borrow, and return books, articles, magazines, journals, et cetera.
-Librarians can add/delete/modify documents and manage copies.

Test cases:
Member: ID - 1, password - 123
Librarian: ID - 2, passwrod - 456
